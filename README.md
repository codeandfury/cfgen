# Cfgen

Code and Fury Rails Generators. Creates templates with commenting on layout and design.

## Installation

Add this line to your application's Gemfile:

    gem 'cfgen'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cfgen
